# Sprint 02 - Windows 11 Endpoint

## Objetivo

Implantar um endpoint Windows 11 dentro do Cybersecurity Home Lab para simular uma estação de trabalho corporativa.

## Atividades realizadas

- Instalação do Windows 11 Pro
- Configuração de conta local
- Integração com rede do pfSense
- Validação de conectividade
- Testes de DNS
- Atualizações do Windows Update

## Evidências

### Configuração de rede

- IP: 192.168.100.100
- Gateway: 192.168.100.1
- DNS: pfSense

### Testes realizados

- Ping para gateway
- Ping para Internet
- Resolução DNS utilizando nslookup

## Skills desenvolvidas

- VMware Workstation
- Windows Administration
- TCP/IP
- DNS
- Troubleshooting
- Network Validation
