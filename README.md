# Cybersecurity Home Lab

Projeto pessoal focado em desenvolvimento prático de competências em:

- Redes
- Firewall
- Hardening
- SIEM
- EDR
- Vulnerability Management
- Threat Hunting
- Incident Response

## Infraestrutura

Host:
- Intel i9-13900K
- 64 GB RAM
- VMware Workstation Pro

Rede do Laboratório:
- 192.168.100.0/24

Máquinas:

- pfSense
- Windows 11
- Ubuntu Server
- Kali Linux
- Metasploitable 2
- Wazuh

## Objetivo

Construir um ambiente semelhante a uma infraestrutura corporativa para estudo de Cybersecurity e Blue Team.
